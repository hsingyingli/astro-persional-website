## Set Interface
|operation| meaning|
|-|-|
|build(A)|given am iterable A, build sequence from items in A|
|len()| number of stored items|
|find(k)|return the stored item with key k|
|inset(x)| add x to set (replace item with key x.key if one already exists)|
|delete(k)| remove and return the stored item with key k|
|find_min()| return the stored item with smallest key|
|find_max()| return the stored item with largest key|
|find_prev(k)|return the stored item with smallest key smaller than k|
|find_next(k)| return the stored item with smallest key larger than k|

## Data structure

|DS|Array|Sorted array|
|-|-|-|
|build(A)|o(n)|o(nlogn)|
|find(k)|o(n)|o(logn)|
|insert(x)/delete(k)|o(n)|o(n)|
|find_min/max()|o(n)|o(1)|
|find_prev/next(k)|o(n)|o(logn)|
